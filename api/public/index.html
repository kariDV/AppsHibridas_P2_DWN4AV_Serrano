    ¿Qué es API|Boards?

    API|Boards es una API REST diseñada para proporcionar acceso eficiente a una amplia variedad de datos relacionados con juegos eurogames. Construida sobre Node.js, Express y MongoDB, esta API ofrece funcionalidades robustas para consulta, exploración y filtrado de información de juegos.

    ¿Qué puedo hacer?

    1. Consultar juegos: Utiliza endpoints específicos para recuperar información detallada sobre juegos eurogames mediante criterios como título, ID o lista completa.
    2. Explorar categorías: Accede a datos de categorías de juegos eurogames, buscando por nombre o ID, y organiza los juegos según estas categorías.
    3. Ordenar y filtrar resultados: Personaliza tus consultas utilizando opciones de ordenamiento y filtrado para organizar y refinar los resultados según tus preferencias.

    Obtener lista completa de los juegos eurogames

    El endpoint juego trae la lista completa de la coleccion de Juegos Eurogame.

    Get http://apiboards.dev/juego

    filtrar un juego eurogame por su id

    Filtrado de la lista de juego por id. Ejemplo:

    Get http://apiboards.dev/juego/3

    filtrar un juego eurogame por el título

    Filtrado de la lista de eurogames por el titulo. En caso de ser más de una palabra utilizar para el espacio entre las mismas %20

    Get http://apiboards.dev/ juego/?titulo=Love%20Letter

    Ordenamiento (A-Z) de los juegos eurogames

    Ordenamiento de los títulos de los juegos con el parametro ?ordenado=true

    Get http://apiboards.dev/ juego/?ordenado=true

    Paginado de los juegos eurogames

    Se trae la lista en página de 2 juegos, agregando el valor ?pagina=NUMERO DE PÁGINA. Ejemplo:

    Get http://apiboards.dev/ juego/?pagina=3

    Obtener todas las categorías de los juegos

    El endpoint categoria trae la lista completa de la colección de Juegos Eurogame.

    Get http://apiboards.dev/categoria/

    filtrar una categoría por su id

    Filtrado de la lista de juego por id. Ejemplo:

    Get http://apiboards.dev/categoria/4

    filtrar una categoría por su título

    Filtrado de la lista de eurogames por una categoria. Ejemplo:

    Get http://apiboards.dev/juego/?categoria=Estrategia
